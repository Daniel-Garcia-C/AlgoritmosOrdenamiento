Compilación:
 - Para estar en la misma ubicación del archivo utilice el comando en CMD "cd [dirección de la carpeta donde se encuentra el archivo]".
 - Para poder compilar es necesario usar "CMD" o una terminal con Java para compilar el archivo "ImpOrdenamiento.java" en su ubicación actual.
 - Luego se ejecuta el código de la siguiente forma el archivo "javac PriorityQueueTest.java".

Entrada de Datos:
 - Conjunto de datos por defecto llamado "datashed.csv" para el ordenamiento con los algoritmos de ordenamientos.
 - Valor númerico entero para definir el orden de ordenamiento [Ascendente o Descendente].
 - Valor númerico para definir el tipo de dato a ordenar en el "datashed.csv" en las columnas.

Salida de datos:
 - Primera salida: 
    - BinaryInsertionSort_ordenado.csv [Lista ordenada por el algoritmo BinaryInsertionSort].
    - MergeSort_ordenado.csv [Lista ordenada por el algoritmo MergeSort].
    - QuickSort_ordenado.csv [Lista ordenada por el algoritmo QuickSort].
    - RadixSort_ordenado.csv [Lista ordenada por el algoritmo RadixSort].
 - Segunda salida:
    - Metricas.txt [Almacena las métricas de los algoritmos de ordenamiento]
        *Tiempo de ejecución del algoritmo
        *Número de comparaciones
        *Número de intercambios
